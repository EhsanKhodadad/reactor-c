◆ _lf_interruptable_sleep_until_locked()

int _lf_interruptable_sleep_until_locked ( environment_t * env,
instant_t wakeup )

Sleep until an absolute time. Since there is no sleep mode in Patmos, and energy saving is not important for real-time systems, we just used a busy sleep.

Sleep until the given wakeup time. This should not be used directly as it does not account for clock synchronization offsets. See clock.h.

Parameters
wakeupint64_t time of wakeup
Returns
int 0 if successful sleep, -1 if awoken by async event
